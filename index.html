Relational/Comparison Operators

Relational/comparison operators are used to compare two values. The different relational operators are:

  • Greater than: >
  • Less than: <
  • Equal to: ==
  • Not equal to: !=
  • Greater than or equal to: >=
  • Less than or equal to: <=

I am sure you must have used some of these in mathematics. Let’s see how we can use them here in programming.

Operator Description Exmaple 1 Exmaple 2 Exmaple 3
> Greater Than
< Less Than
== Equal To
!= Not Equal To
>= Greater Than Equal To
<= Less Than Equal To
Conditional Programming

Let’s say someone asks you, “Are you going to wear a raincoat today on the way to school?”
Your answer would be either a Yes or No.
Does your answer depend on something?

Yes! It depends on whether it is raining or not.
So obviously if it is raining you will wear a raincoat (unless you want to reach school all drenched!) and if it is not raining you will not wear a raincoat. So the condition here is “Is it raining?”

In programming too, we often have to make important decisions based on certain conditions, and we can do that using if-else statements.

If we take the same example again, if it is raining you will wear a raincoat, else you will not wear a raincoat.
